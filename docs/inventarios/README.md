# Inventarios

Documentación técnica del módulo de inventarios.

Archivos disponibles:

- `arquitectura_servicios.md`: explica la arquitectura interna, servicios, selectors, flujos y reglas de mantenimiento.

Esta documentación corresponde al cierre técnico posterior a la finalización del desarrollo del módulo.
